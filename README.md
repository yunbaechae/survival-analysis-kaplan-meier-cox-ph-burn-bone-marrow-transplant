# Survival Analysis of Burn and Bone Marrow Transplant Data Using Cox PH and Kaplan-Meier Estimators

ACST3058 Actuarial Modelling Assignment 1 (2023)

This assignment applied Kaplan-Meier estimator and Cox proportional hazards model to survival data from burn and bone marrow transplant patients. Focused on survival probability estimation, hazard modelling, and quantitative risk assessment.

- Fitted Cox PH model to estimate hazard ratios for covariates  
- Analyzed survival trends and quantitative implications
